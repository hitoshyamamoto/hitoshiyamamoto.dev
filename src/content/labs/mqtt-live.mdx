Demo `kind: realtime` servido por uma island React local. Por padrão simula um
stream de telemetria no próprio cliente (sem broker, sem cold start). Para ligar
a um broker MQTT real via WebSocket, conecte o componente a um `wss://` —
idealmente atrás de uma função serverless quando precisar esconder credenciais.

Respeita `prefers-reduced-motion`: com movimento reduzido, a animação contínua é
desativada.
